    # js

    • 数组的定义写法正确的是
      • var arr = new Array(1, 2, 3)
      • var arr = new Array(3)
      • var arr = new Array('1', '2')
      • var arr[] = new Array(2)(3)

    # css 定位

    • fixed 始终基于 body 定位吗？
      • position: fixed的元素是相对于最近的具有定位（position）属性的祖先元素进行定位的，而不是始终基于body元素。
      	<!DOCTYPE html>
	<html>
		<body>
			<div id="container">
				<div id="fixed">
					fixed
				</div>
			</div>
		</body>
		<style>
			#container {
				width: 200px;
				height: 200px;
				background-color: red;
				position: absolute;
				bottom:0;
			}

			#fixed {
				position: fixed;
				right: 0;
				width: 50px;
				height: 50px;
				background-color: antiquewhite;
			}
		</style>
	</html>

    • relative 相对定位，移动后，不再占据原来的位置?
      • 相对定位的元素，移动后，仍然占据原来的位置，只是视觉上移动了，不会影响其他元素的位置。

    1 2 3

    2. 子组件检查输入的数字是否合法 合法范围为[0, 10], 如果不合法，将父组件中的背景颜色设置为红色
    3. 子组件渲染表格
