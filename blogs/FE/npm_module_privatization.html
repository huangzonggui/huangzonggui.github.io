    在第三方模块中修改使用

    Aki Huang 12/4/2021 npm

    # 场景

    • 业务中需要的组件，现有的组件中无法满足，需要稍作修改
    • 为了沉淀组件，做到复用，需要在现有的组件中修改

    # 步骤

    • 在github上将代码fork一份
    • 下载到本地
    • 修改，编译
    • 发版（patch、minor、major）
      • 自动生成版本号
      • 自动将commit 的 message 提交到 change log中
      • 持续集成
        • 利用Github Actions自动发布，CI持续集成 (opens new window)
          • Github Actions 入门教程 (opens new window)
        • 持续集成服务 Travis CI 教程 (opens new window)
